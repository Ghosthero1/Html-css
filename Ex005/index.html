Unordered lists(Listas não Ordenadas) são listas com marcadores, que são aquelas onde a ordem dos itens não influência

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Mudando o marcadores

  • disc:padrão. Uma bola preta totalmente pintada
    • cicle:Uma bola com uma borda preta e sem preenchimento
    • square:Um pequeno preto totalmente pintado

É como se fose um diionário, toda lista de definições está dentro de uma tag dl(definiton list) cada termo é um dt(definition term) e cada descrição é um dd(definiton description).
